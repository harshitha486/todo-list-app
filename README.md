# To-Do List App

A simple and responsive To-Do List application built using HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Mark tasks as completed
- Remove tasks
- Tasks stored using Local Storage
- Responsive and clean UI

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How It Works
- Users can type a task and add it to the list
- Clicking on a task marks it as completed
- Clicking the × icon removes the task
- Tasks are saved in the browser using Local Storage, so they remain even after refreshing the page
